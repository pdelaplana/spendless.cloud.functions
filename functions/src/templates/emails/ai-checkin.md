## Technical Notes

### Variables to Replace
- `{userName}` - User's name from account profile
- `{periodName}` - Name of the period being analyzed (e.g., "January 2025", "Week of Jan 1")
- `{insights}` - The formatted AI insights (markdown converted to HTML)

### Send Timing
- Trigger: After AI analysis completes successfully
- Weekly: Every Monday morning (for users with weekly preference)
- Period End: When a spending period ends (for users with period-end preference)

### Features
- Premium feature only
- Users can configure frequency (weekly, period-end, both, or off)
- Insights include spending patterns, category breakdown, tag analysis, and recommendations

### Styling Guidelines
- Use Spendless brand colors
- Mobile-responsive design
- Ensure insights section is clearly formatted with proper spacing
- Use icons/emojis to make sections more engaging
- Keep overall tone encouraging and helpful, not judgmental
